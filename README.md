
# ✊✋✌️ Stone Paper Scissors Game

A fun and interactive **Stone, Paper, Scissors** game built with HTML, CSS, and JavaScript — playable right in your browser!

---

## 🎮 Demo

> Play against the computer and see if you can beat it!

---

## 🚀 Features

- 🤖 Play against a computer with random moves
- 🏆 Score tracking for Player and Computer
- 🎨 Clean and responsive UI
- ⚡ Instant result display after each round
- 🔄 Reset/Restart game functionality

## ▶️ How to Play

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BOT9315/Stone-Paper-Scissors-game-.git
   ```

2. **Navigate into the project folder:**
   ```bash
   cd Stone-Paper-Scissors-game-
   ```

3. **Open `index.html` in your browser:**
   ```bash
   # On macOS
   open index.html

   # On Windows
   start index.html

   # On Linux
   xdg-open index.html
   ```

4. Click **Stone**, **Paper**, or **Scissors** to make your move.
5. The computer will randomly pick its move.
6. The winner of each round is displayed instantly!

---

## 🎯 Game Rules

| Player | Computer | Result |
|--------|----------|--------|
| ✊ Stone | ✌️ Scissors | Player Wins |
| ✋ Paper | ✊ Stone | Player Wins |
| ✌️ Scissors | ✋ Paper | Player Wins |
| Any | Same | Draw |
